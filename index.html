🌽cmd 命令

查所有端口开没开 netstat -ano -b

查所有进程 tasklist

查看某个端口对应的进程 netstat -ano | findstr :8080

查看 pid 进程名称 tasklist | findstr 4444

导出所有进程到 CSV 文件 tasklist /fo csv > processes.csv

列出所有 Java 进程并显示 PID tasklist | findstr java

按条件过滤进程（如 imagename, pid） tasklist /fi "imagename eq chrome.exe" tasklist /fi "pid eq 4444"

显示进程关联的 Windows 服务 tasklist /svc
